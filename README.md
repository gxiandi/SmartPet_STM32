# STM32仿生交互桌面宠物机器人

## 项目简介
基于STM32F103C8T6搭建的仿生交互桌面机器人，通过舵机驱动实现多动作仿生效果，并结合语音与蓝牙交互，打造具有情感反馈的桌面宠物系统。

## 核心功能
- 多舵机动作控制：通过TIM定时器输出5路PWM信号，实现舵机角度精准控制（精度±1°），完成前进、跳跃、摇尾巴等16种仿生动作
- OLED表情反馈：采用I2C驱动0.96寸OLED显示屏，实现双缓存显示机制，表情切换零卡顿、无闪烁
- 蓝牙+语音交互：通过双串口接收指令，设计全局状态机管理动作优先级，实现指令优先调度与多状态交互
- 系统稳定性：采用模块化分层架构，底层驱动与业务逻辑解耦，系统运行无冲突、无死机，指令响应延迟<100ms

## 技术栈
- 主控：STM32F103C8T6
- 外设：舵机、0.96寸OLED显示屏、蓝牙模块、语音模块
- 协议：UART、I2C、PWM
- 开发工具：Keil MDK、STM32CubeIDE

## 项目亮点
- 多模态指令控制与情感表情反馈，交互性强，仿生效果生动
- 解决多指令并发与动作抖动问题，系统响应快、稳定性高
- 为仿生机器人与嵌入式综合开发提供了完整的实践原型
