# STM32仿生交互桌面宠物机器人

## 项目简介
基于 STM32F103C8T6 主控搭建的仿生交互桌面机器人项目，通过多路 PWM 精准控制舵机实现仿生动作，结合蓝牙 + 语音双模指令交互与 OLED 情感表情显示，打造低延迟、高稳定性、沉浸式交互的桌面宠物系统。
项目涵盖 STM32 外设驱动、多串口通信、状态机调度、模块化架构设计，是嵌入式软硬件协同开发的综合实践。

## 硬件清单
- 主控：STM32F103C8T6
- 执行器：5路模拟舵机
- 显示模块：0.96寸 I2C OLED 屏
- 通信模块：蓝牙模块、语音识别模块
- 辅助：LED 指示灯、杜邦线、5V 电源

## 核心功能
- 多舵机动作控制：通过 TIM 定时器输出 5 路 PWM 信号，实现舵机角度精准控制（精度 ±1°），完成前进、跳跃、摇尾巴等 16 种仿生动作
- OLED 表情反馈：采用 I2C 驱动 0.96 寸 OLED，实现双缓存显示机制，6 种动态表情切换零卡顿、无闪烁
- 蓝牙 + 语音双模交互：双串口中断接收指令，全局状态机统一管理动作模式，支持指令优先级调度
- 系统稳定可靠：模块化分层架构，驱动层与业务层解耦，系统运行无冲突、无死机，指令响应延迟 <100ms

## 技术栈
- 主控：STM32F103C8T6
- 开发库：STM32 标准库
- 通信协议：UART、I2C、PWM
- 软件架构：全局状态机、模块化分层设计
- 开发工具：Keil MDK、逻辑分析仪

## 项目结构
├── Drivers/ 底层驱动：GPIO、UART、TIM、PWM、I2C、OLED
├── Hardware/ 硬件模块驱动：舵机、蓝牙、语音
├── Application/ 应用层：状态机、动作解析、指令调度
├── Main.c / 主函数、系统初始化、主循环


## 快速开始
1. 克隆工程，使用 Keil MDK 打开
2. 编译代码并下载至 STM32F103C8T6
3. 连接舵机、OLED、蓝牙模块、语音模块
4. 上电运行，通过语音或蓝牙发送指令即可触发动作与表情

## 关键技术实现
- 多路 PWM 精准控制：50Hz 舵机驱动，1us 分辨率，角度误差 ±1°
- 双串口中断：独立接收蓝牙与语音指令，环形缓冲区防止数据丢失
- 状态机调度：实现多动作互斥、优先级管理，避免动作冲突
- OLED 双缓存：预加载下一帧画面，实现无闪烁、零卡顿切换

## 解决的关键问题
- 舵机抖动：通过软件缓动 + 供电滤波优化，动作平稳无抖动
- 双串口数据丢包：中断优化 + 数据校验，实现稳定通信
- 多指令冲突：状态机 + 优先级调度，保证系统流畅运行

## 项目亮点
- 多模态指令控制 + 情感表情反馈，交互体验强
- 高精度运动控制 + 低延迟响应，系统性能优异
- 模块化架构清晰，可移植性强，便于功能扩展
- 完整覆盖嵌入式开发全流程，适合学习与面试展示
